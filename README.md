# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/d7535ede-66c7-497e-a169-bbca14050599

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key
3. (Optional) Set `GEMINI_LIVE_MODEL` in `.env.local` if you want to override the default live model (`gemini-live-2.5-flash-preview`)
4. Run the app:
   `npm run dev`
